## Post #1 by @system (2017-04-07 17:17 UTC)

<p>This is a permanent topic, visible only to staff, for storing images and files used in the site design. Don’t delete it!</p>
<p>Here’s how:</p>
<ol>
<li>Reply to this topic.</li>
<li>Upload all the images you wish to use for logos, favicons, and so forth here. (Use the upload toolbar icon in the post editor, or drag-and-drop or paste images.)</li>
<li>Submit your reply to post it.</li>
<li>Right click the images in your new post to get the path to the uploaded images, or click the edit icon to edit your post and retrieve the path to the images. Copy the image paths.</li>
<li>Paste those image paths into <a href="/admin/site_settings/category/required">basic settings</a>.</li>
</ol>
<p>If you need to enable different file type uploads, edit <code>authorized_extensions</code> in the <a href="/admin/site_settings/category/files">file settings</a>.</p>
