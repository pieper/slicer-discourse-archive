## Post #2 by @lassoan (2017-08-14 12:12 UTC)

<p>Have you build Slicer yourself? If yes, and you need to use Slicer’s appstore for getting extensions, then you must use exactly the same build settings as the Slicer factory machines (most importantly, use SVN repository and a version for that factory machines created extensions).</p>
<p>Note that today there was an error with extension builds on factory machines - see <a href="https://discourse.slicer.org/t/no-extensions-for-latest-nightly/865">No extensions for latest nightly</a></p>
