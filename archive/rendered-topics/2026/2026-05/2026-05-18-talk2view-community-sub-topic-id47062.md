## Post #1 by @benzwick (2026-05-18 00:51 UTC)

<p>Is there any chance we could make a sub-topic for Talk2View?</p>
<p>Talk2View, “the medical viewer you can talk to”, is available as a 3D Slicer Extension or as a 3D Slicer Custom Application. We also have an OHIF version at <a href="https://app.talk2view.com/" rel="noopener nofollow ugc">https://app.talk2view.com/</a> as well as a Talk2View SDK that allows anyone to add voice-controlled agentic AI to any website or application (you can try it out by using it to navigate our new website by clicking on the Talk2View icon in the lower right corner: <a href="https://talk2view.com/" rel="noopener nofollow ugc">https://talk2view.com/</a>).</p>
<p>It would be great to have a separate sub-topic for Talk2View so that we can put all our Slicer related announcements in one place. We are also planning to open source some of the Slicer components so it would be a great place to discuss that as well.</p>
<p>For those who haven’t tried it out yet, you can download the Talk2View 3D Slicer Extension here: <a href="https://talk2view.com/products/integration" class="inline-onebox" rel="noopener nofollow ugc">Integrations | Talk2View</a></p>
