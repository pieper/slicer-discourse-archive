## Post #1 by @Thibault_Pelletier (2026-05-21 20:30 UTC)

<p>We are pleased to announced a series of recent <code>trame-slicer</code> additions which have added significant improvements to segmentation, application structure and core compatibility.</p>
<ul>
<li><strong>Standalone Version</strong>: Trame slicer can now be fully pip installed outside 3D Slicer using the <code>pip install "trame-slicer[standalone]"</code></li>
<li><strong>Standalone Applications</strong>: The examples have been promoted to first-class applications. They now can be run using python module syntaxe: <code>python -m trame_slicer.app.segmentation_app</code></li>
<li><strong>Segmentation</strong>: We have brought a comprehensive suite of familiar 3D Slicer segmentation tools to the web, including Draw segmentation, Scissors, logical operators, smoothing, and volume intensity masking.</li>
</ul>
<p>For a full list of changes and technical details, please refer to the <a href="https://github.com/KitwareMedical/trame-slicer/blob/main/CHANGELOG.md" rel="noopener nofollow ugc">CHANGELOG.md</a>.</p>

Thibault</p>
