## Post #2 by @aiden.zhu (2026-07-21 17:54 UTC)

<p>It seems something happened to pytorch? Maybe.</p>
<p>Option-A:<br>
Reinstall PyTorch via Slicer’s PyTorch Extension<br>
Open 3D Slicer.<br>
Go to the PyTorch module ( search for it in the module dropdown).<br>
Click Uninstall PyTorch (or force reinstall if the option exists).<br>
Click Install PyTorch (select CUDA if you have a compatible NVIDIA GPU, or CPU otherwise).<br>
Restart 3D Slicer and try running TotalSegmentator again.</p>
<p>Option-B:<br>
Clean the Python Environment manually<br>
If the method-A doesn’t work, clean up the corrupted package from Slicer’s internal Python console:<br>
Open 3D Slicer.<br>
Open the Python Console (View → Python Console, or press Ctrl + 3).<br>
Run the following lines:<br>
import slicer<br>
slicer.util.pip_install(“–force-reinstall light-the-torch”)<br>
slicer.util.pip_install(“–force-reinstall torch torchvision torchaudio”)<br>
then restart slicer</p>
<p>Good luck.</p>
