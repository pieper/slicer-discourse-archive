<h2><a name="p-132538-slicer-core-thibault-6" class="anchor" href="#p-132538-slicer-core-thibault-6" aria-label="Heading link"></a>slicer-core (Thibault)</h2>
<p>A big milestone has been reached into making the Slicer core libraries available on PyPI.<br>
A preview version has been pushed to PyPI for Slicer version 5.11.0.2 and is compatible with Windows / Linux &amp; MacOS for Python version 3.10 up to 3.13.</p>
<p>In the next weeks, we will work on:</p>
<ul>
<li>Having the SlicerCore-SDK available to create extensions compatible with SlicerCore</li>
<li>Update the SlicerLayerDM extension to be available on PyPI (required by trame-slicer)</li>
<li>Update trame-slicer to be compatible with the PyPI versions of SlicerCore and SlicerLayerDM</li>
<li>Documenting the SlicerCore project, how to extend it and more</li>
</ul>
