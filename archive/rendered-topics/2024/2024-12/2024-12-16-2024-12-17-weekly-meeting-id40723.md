## Post #2 by @jamesobutler (2024-12-17 03:44 UTC)

<p>I would like to discuss the Slicer GitHub organization housing the <a href="https://github.com/pmeier/light-the-torch" rel="noopener nofollow ugc"><code>light-the-torch</code></a> repository to facilitate continued maintenance and support by the Slicer community. This python package is utilized by the <a href="https://github.com/fepegar/SlicerPyTorch" rel="noopener nofollow ugc">SlicerPyTorch</a> extension which is currently used by 14 Slicer extensions and growing. The original author of <code>light-the-torch</code> has acknowledged that the python package has become a bit of a burden and that he is no longer an active user of the python package. He is willing to transfer the repository to the Slicer community so that Slicer developers can lead the maintenance for the project. Details are available at <a href="https://github.com/pmeier/light-the-torch/issues/152" class="inline-onebox" rel="noopener nofollow ugc">light-the-torch future maintenance discussion · Issue #152 · pmeier/light-the-torch · GitHub</a>.</p>
