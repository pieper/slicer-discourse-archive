## Post #4 by @mikebind (2024-05-22 19:21 UTC)

<p>When you click on it in SampleData, you download it, so you already have it. If you click “Save”, you can choose what format you want to save the image it in the save scene dialog window.  Or, you can go to the Data module, right click on the entry for the image you want, and select “Export to file” to save only the image volume without saving the scene file as well.</p>

## Post #8 by @DaniTheStudent (2024-06-01 16:34 UTC)

<p>NVM i have finally done it!</p>
<p>Here is the python code that i had to introduce in the console</p>
<p>import os<br>
import slicer<br>
import vtk<br>
from vtk.util.numpy_support import numpy_to_vtk</p>
<h1><a name="p-112129-define-the-output-directory-1" class="anchor" href="#p-112129-define-the-output-directory-1" aria-label="Heading link"></a>Define the output directory</h1>
<p>outputDir = r’G:/Kaggle imagini/Extract test din 3d slicer/MRBrainTumor1_Images’<br>
if not os.path.exists(outputDir):<br>
os.makedirs(outputDir)</p>
<h1><a name="p-112129-load-the-sample-data-2" class="anchor" href="#p-112129-load-the-sample-data-2" aria-label="Heading link"></a>Load the sample data</h1>
<p>import SampleData<br>
sampleDataLogic = SampleData.SampleDataLogic()<br>
mrBrainTumorVolume = sampleDataLogic.downloadSample(‘MRBrainTumor1’)</p>
<h1><a name="p-112129-get-the-volume-node-3" class="anchor" href="#p-112129-get-the-volume-node-3" aria-label="Heading link"></a>Get the volume node</h1>
<p>volumeNode = slicer.util.getNode(‘MRBrainTumor1’)</p>
<h1><a name="p-112129-get-the-volume-array-4" class="anchor" href="#p-112129-get-the-volume-array-4" aria-label="Heading link"></a>Get the volume array</h1>
<p>volumeArray = slicer.util.arrayFromVolume(volumeNode)</p>
<h1><a name="p-112129-save-each-slice-as-a-jpg-file-5" class="anchor" href="#p-112129-save-each-slice-as-a-jpg-file-5" aria-label="Heading link"></a>Save each slice as a JPG file</h1>
<p>numSlices = volumeArray.shape[0]<br>
for sliceIndex in range(numSlices):<br>
sliceArray = volumeArray[sliceIndex, :, :]</p>
<h1><a name="p-112129-flip-the-array-vertically-6" class="anchor" href="#p-112129-flip-the-array-vertically-6" aria-label="Heading link"></a>Flip the array vertically</h1>
<p>sliceArray = sliceArray[::-1, :]</p>
<h1><a name="p-112129-convert-the-numpy-array-to-a-vtkimagedata-7" class="anchor" href="#p-112129-convert-the-numpy-array-to-a-vtkimagedata-7" aria-label="Heading link"></a>Convert the numpy array to a vtkImageData</h1>
<p>vtkImageData = vtk.vtkImageData()<br>
vtkImageData.SetDimensions(sliceArray.shape[1], sliceArray.shape[0], 1)<br>
vtkImageData.AllocateScalars(vtk.VTK_UNSIGNED_CHAR, 1)<br>
sliceArrayVtk = numpy_to_vtk(sliceArray.ravel(), deep=True, array_type=vtk.VTK_UNSIGNED_CHAR)<br>
vtkImageData.GetPointData().SetScalars(sliceArrayVtk)</p>
<h1><a name="p-112129-write-the-slice-to-a-jpg-file-8" class="anchor" href="#p-112129-write-the-slice-to-a-jpg-file-8" aria-label="Heading link"></a>Write the slice to a JPG file</h1>
<p>writer = vtk.vtkJPEGWriter()<br>
sliceFilename = os.path.join(outputDir, f’MRBrainTumor1_slice{sliceIndex:03}.jpg’)<br>
writer.SetFileName(sliceFilename)<br>
writer.SetInputData(vtkImageData)<br>
writer.Write()</p>
<p>print(“Export completed.”)</p>
