<p>Some of the differences from Jupyter:</p>
<ul>
<li>Fully reactive: change a variable value and all cells using that value will update. Observablehq uses this reactive model and it works quite well.</li>
<li>Notebook format is python code, not JSON, so notebooks are more git friendly and reusable.</li>
</ul>
