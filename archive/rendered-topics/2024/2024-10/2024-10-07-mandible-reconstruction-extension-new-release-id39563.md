Also, the GUI has been refactored a bit, adding some icons to improve user experience. And there are buttons that let you open the documentation, report a bug or request a feature with one click.</p>
<p>Boolean operations are now more robust and Slicer does not crash anymore for this cause.</p>
<p>An inter-condylar beam can be added to the neomandible reconstruction 3D model so it’s more rigid after 3D printing</p>
<p>The Virtual Surgical Plan can be locked, so you can avoid changing the plan by mistake once it’s finished</p>
<p>The <a href="https://github.com/SlicerIGT/SlicerBoneReconstructionPlanner#instructions" rel="noopener nofollow ugc">text instructions</a> have been updated to the changes of this version</p>
