## Post #4 by @pieper (2023-04-25 22:57 UTC)

<p>Unfortunately it’s due to some change in VTK (the underlying graphics layer) that changed sometime after the feature was added to Slicer.  It’s been like this for a while and I don’t know of anyone actively funded to work on it so there’s no estimated fix.  Learning to use python really isn’t that hard, but of course that depends on how much you need to get the high res image.  Maybe you can find a colleague to help you.</p>
