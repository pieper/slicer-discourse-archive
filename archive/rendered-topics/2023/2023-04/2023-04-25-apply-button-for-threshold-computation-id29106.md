## Post #2 by @lassoan (2023-04-25 11:21 UTC)

<p>Click “Show details” to get help on how to use “Local Threshold” effect. As far as I remember, you need to Ctrl-click in the image where you want to apply the local thresholding.</p>
<p>If you want to do simple Global thresholding then you can use the  “Threshold” effect.</p>
