## Post #1 by @jcfr (2023-03-23 12:37 UTC)

<p>You may install the <strong>Kitware File Viewers</strong> extension from one of the following web stores:</p>
<ul>
<li><a href="https://chrome.google.com/webstore/detail/kitware-file-viewers/lklknpjabnmfefcfhmhdnhebkpindcld">Chrome</a></li>
<li><a href="https://addons.mozilla.org/en-US/firefox/addon/kitware-file-viewers/">Firefox</a></li>
<li><a href="https://microsoftedge.microsoft.com/addons/detail/kitware-file-viewers/kfejllcdobijgoijjpfcbaoicjbdkhfb">Edge</a></li>
</ul>
<p>To learn more:</p>
<ul>
<li><a href="https://www.kitware.com/kitware-file-viewers-browser-add-on/">https://www.kitware.com/kitware-file-viewers-browser-add-on/</a></li>
<li><a href="https://github.com/KitwareMedical/KitwareFileViewers" class="inline-onebox">GitHub - KitwareMedical/KitwareFileViewers: Browser extension to open URL in Kitware online viewers</a></li>
</ul>
