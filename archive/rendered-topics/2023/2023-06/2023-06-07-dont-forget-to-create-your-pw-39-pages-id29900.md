## Post #1 by @Sam_Horvath (2023-06-07 15:11 UTC)

<p>If you are attending PW 39, in person or remote, please don’t forget to get started on your <a href="https://projectweek.na-mic.org/PW39_2023_Montreal/#projects">project page</a>!</p>
<h3><a name="p-95921-project-creation-tips-1" class="anchor" href="#p-95921-project-creation-tips-1" aria-label="Heading link"></a>Project Creation Tips</h3>
<ul>
<li>
<p>Get your project pages created early!  The day before is best to make sure everything you need for you presentation is available.  The Project Week site will be closed to edits for the <em><strong>10 minutes before</strong></em> both the opening and closing presentation session to ensure site generation. After this 10 minute period edits will be re-enabled.</p>
</li>
<li>
<p>If you are <a>creating the project page yourself using the template</a>, <strong>don’t reuse a project page template from a previous year.</strong>  We have made significant updates to the template to support auto-generation of project pages, so previous years’ templates will not function properly.</p>
</li>
<li>
<p>Remember to fill out the title for your project when using the <a href="https://github.com/NA-MIC/ProjectWeek/issues/new?assignees=drouin-simon%2Cpiiq%2Crafaelpalomar%2Csjh26%2Ctkapur&amp;labels=project%2Cevent%3APW39_2023_Montreal&amp;projects=&amp;template=project.yml&amp;title=Project%3A+">project creation issue</a></p>
</li>
<li>
<p>Check the formatting on the Key Investigators list when creating a <a href="https://github.com/NA-MIC/ProjectWeek/issues/new?assignees=drouin-simon%2Cpiiq%2Crafaelpalomar%2Csjh26%2Ctkapur&amp;labels=project%2Cevent%3APW39_2023_Montreal&amp;projects=&amp;template=project.yml&amp;title=Project%3A+">project issue</a> (this is critical for page generation):</p>
<p><code>- Firstname Lastname (Affiliation, Country)</code></p>
</li>
</ul>
