## Post #2 by @lassoan (2023-08-02 15:02 UTC)

<p>Unfortunately, this can happen time to time with executable files that are changed frequently (we upload a new Slicer Preview Release every day), because malware detection engines are tuned continuously and may make false detections for files that they have not seen before. What web browser do you use?</p>
<p>Typically all you need is to wait some time (half day or maybe a day) and retry, because many users report such errors to the developer to the anti-malware tool and they make quick adjustments. If the problem persists for more than a day then let us know and we report the download as safe to the browser/anti-malware tool developer.</p>
<p>In the meantime you can use a different browser or different computer to download the file.</p>
