## Post #2 by @lassoan (2018-09-16 13:41 UTC)

<p>RealSense device can be connected to Slicer via <a>Plus toolkit</a>. It is just yet another imaging device, so any tutorial that includes any imaging device is applicable. See specification of RealSense-specific options and example configuration file that was used for creating the referenced youtube video is available in <a href="http://perk-software.cs.queensu.ca/plus/doc/nightly/user/DeviceIntelRealSense.html">Plus toolkitâ€™s user manual</a>.</p>
