## Post #2 by @lassoan (2018-09-29 13:13 UTC)

<p>All clinical scanners save CT volumes in DICOM format. Slicer can load these.</p>
<p>Slicer can also load image data from a wide range of other file formats - see complete list here: <a href="https://www.slicer.org/wiki/Documentation/Nightly/SlicerApplication/SupportedDataFormat">https://www.slicer.org/wiki/Documentation/Nightly/SlicerApplication/SupportedDataFormat</a></p>
