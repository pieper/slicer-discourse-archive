## Post #2 by @lassoan (2018-07-22 02:36 UTC)

<p>Slicer allows you unlimited comparison of various data sets. If you only want to see one data set then close the scene before loading a new data set (hit Ctrl+W or in menu: File / Close scene) .</p>
