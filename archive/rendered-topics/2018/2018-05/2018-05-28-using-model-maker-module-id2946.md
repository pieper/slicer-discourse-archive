## Post #4 by @lassoan (2018-05-28 05:29 UTC)

<p>Since Segmentations and Segment Editor modules have been introduced, I would not recommend using model maker module.</p>
<p>You can <a href="https://www.slicer.org/wiki/Documentation/Nightly/Training#Slicer4_Image_Segmentation">create nice segmentations using Segment editor</a> module and <a href="https://discourse.slicer.org/t/save-segmentation-directly-to-stl-or-obj-files/2428">export the result directly to STL/OBJ files</a>.</p>
