<li>Install PureData   -   How to install pureData ???</li>
<li>Install 3D Slicer -  OK</li>
<li>Start 3D Slicer -  OK</li>
<li>Install the SoundControl extension - OK</li>
<li>Install SlicerIGT extension (it contains BreachWarning module, which is used for computing distance of tooltip from a target model) - OK</li>
</ul>
<p>Setup:</p>
<ul>
<li>Start 3D Slicer</li>
<li>Load BreachWarningScene.mrb</li>
<li>Switch to “Open Sound Control module”</li>
<li>In PureData section, select OscSimpleTest.pd configuration file, click “Start server” button</li>
<li>Switch to “Sound Navigation” module</li>
<li>Move the needle model by shift+left click &amp; drag =&gt; moving the needle closer to the target will increase beepinthe g frequency</li>
</ul>
