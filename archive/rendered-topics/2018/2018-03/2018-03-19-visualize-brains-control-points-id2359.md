## Post #11 by @lassoan (2018-03-19 14:14 UTC)

<p>Transform visualization does not deal with internal details of transforms. If you want to see the control points, probably the easiest is to write a few-line Python script that places a markup fiducial at each control point (you can compute it from the B-spline grid size, origin, spacing, and orientation). You can then apply the transform to these markups to see the transformed points.</p>
<p>If you use thin-plate-spline transform instead of b-spline then you can also edit transforms in real-time by drag-and-dropping the markup fiducial points (using SlicerIGT extension’s Fiducial registration wizard module).</p>
