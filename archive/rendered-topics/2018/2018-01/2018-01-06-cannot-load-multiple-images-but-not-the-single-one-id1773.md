## Post #2 by @pieper (2018-01-06 15:19 UTC)

<p>When you don’t have the Single File option checked then the file reader will try to look for patterns of similarly named files (like img1.png, img2.png, img3.png…) and try to assemble a volume.  But if the files don’t match, like they are different sizes, then the reader can fail.</p>
