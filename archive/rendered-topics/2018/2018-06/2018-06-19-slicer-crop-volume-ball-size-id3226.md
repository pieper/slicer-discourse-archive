## Post #2 by @lassoan (2018-06-19 13:00 UTC)

<p>Ball size was defined in pixels in Slicer-4.8.1, which made it too small on high-resolution screens. The issue is fixed in recent nightly versions of Slicer, where the size is defined relative to screen size.</p>
