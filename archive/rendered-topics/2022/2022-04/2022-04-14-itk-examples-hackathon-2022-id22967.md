## Post #1 by @tom.birdsong (2022-04-14 16:14 UTC)

<p>The <a href="https://github.com/insightsoftwareconsortium/itk" rel="noopener nofollow ugc">Insight Toolkit (ITK)</a> is an open-source, cross-platform toolkit for N-dimensional scientific image processing, segmentation, and registration. Documentation for the toolkit ranges from the <a href="https://itk.org/ItkSoftwareGuide.pdf" rel="noopener nofollow ugc">ITK Software Guide</a> textbook, to <a href="https://itk.org/Doxygen/html/index.html" rel="noopener nofollow ugc">Doxygen API reference material</a>, to <a href="https://examples.itk.org/index.html" rel="noopener nofollow ugc">C++ and Python examples</a>. Slicer makes use of ITK filters for many image processing operations.</p>
<p>The ITK Examples hackathon aims to bring together ITK developers and new users to expand documentation and learn to use ITK tools for scientific image processing.</p>
<p>We will focus on expanding and adding new examples to <a href="https://examples.itk.org/index.html" rel="noopener nofollow ugc">https://examples.itk.org/index.html</a>.</p>
<p>More information may be found at <a href="https://hackmd.io/WL6AxKj6Sy2O9GKtgj0LYA?view" rel="noopener nofollow ugc">https://hackmd.io/WL6AxKj6Sy2O9GKtgj0LYA?view</a>.</p>
<h2><a name="p-76770-why-attend-1" class="anchor" href="#p-76770-why-attend-1" aria-label="Heading link"></a>Why Attend?</h2>
<ul>
<li>Learn: Want to learn to use ITK, but not sure where to get started? Join the discussion and take your first steps with the ITK Python library!</li>
<li>Contribute: Have you found a great way to apply ITK in your image processing work? Create an example to share your process with other community members!</li>
<li>Meet: Want to interact with the ITK and scientific image processing community? Come meet the developers behind the curtain!</li>
</ul>
