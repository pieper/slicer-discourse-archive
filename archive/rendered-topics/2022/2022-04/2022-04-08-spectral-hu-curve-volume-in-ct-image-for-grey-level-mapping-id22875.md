## Post #11 by @lassoan (2022-04-17 03:46 UTC)

<p>Yes, you should be able to make such plots. You need to segment these structures in the images (at each energy). I’m not sure how the mg/cm3 values are computed, but you can read up on it and see how trivial the computation is. Getting the statistics and plotting the results should be no problem - you can use Slicer’s Plots module or matplotlib (or any other Python plotting package).</p>
