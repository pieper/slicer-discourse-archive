# Job position: Software Engineer @ SonoVol, Inc.

## Post #1 by @tomekcz (2022-05-20 05:36 UTC)

<p><strong>Position Description:</strong> We are seeking two early-career developers who are excited about the opportunity to create software for our <em>in vivo</em> 3D imaging tools. In the role of Software Engineer, your responsibilities will include:</p>
<ul>
<li>
<p><strong>Software development.</strong> Develop front and back-end code to both integrate software subsystems with acquisition system hardware (including ultrasound scanners, scan guidance camera, and robotic motion stages) and analyze ultrasound data.</p>
</li>
<li>
<p><strong>GUI design.</strong> Design and program front-end graphical user interface (GUI) elements with streamlined user experience.</p>
</li>
<li>
<p><strong>Project management.</strong> Help establish estimates for project timelines. Diligently record progress via the team’s project management software. Work in a collaborative fashion with both hardware and software teams to meet company objectives.</p>
</li>
<li>
<p><strong>Testing and code review.</strong> Write and execute verification and validation tests of various software modules. Document code and conduct code reviews.</p>
</li>
<li>
<p><strong>Technical support.</strong> Field and document technical support inquiries from customers.</p>
</li>
</ul>
<p><strong>Company Overview:</strong></p>
<p>SonoVol (<a href="http://www.sonovol.com" rel="noopener nofollow ugc">www.sonovol.com</a>), a PerkinElmer company, is focused on improving ultrasound imaging approaches for disease research and drug testing at the preclinical stage. SonoVol has launched and continually develops the Vega® imaging platform, the first robotically controlled preclinical ultrasound imaging tool of its kind. Using 3D Slicer as a backbone, SonoVol has created SonoEQ™, a full software solution for 3D data acquisition and analysis that can enable high-throughput multi-modality 3D image capture. SonoVol was formed in 2014 as a spin-out from the Joint Department of Biomedical Engineering at The University of North Carolina at Chapel Hill and North Carolina State University, and is headquartered in Durham, NC, USA.</p>
