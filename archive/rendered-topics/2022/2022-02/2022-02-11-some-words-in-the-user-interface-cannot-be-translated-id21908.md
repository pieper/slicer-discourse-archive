## Post #2 by @lassoan (2022-02-11 05:47 UTC)

<p>Strings provided by Python scripted and CLI modules, strings that are dynamically added by C++ loadable modules and not marked as translatable, and strings in extensions are not available for translation yet. See a bit more information here: <a href="https://github.com/Slicer/Slicer/issues/6177">https://github.com/Slicer/Slicer/issues/6177</a></p>
<p>We are in the process of hiring an engineer dedicated to internationalization (funded by CZI Essential Open-Source Software grant), so it will probably take a few months until all these missing strings become available for translation. In the meantime, if any software developer had time to help out with these tasks then we would be very happy to collaborate.</p>
