## Post #2 by @jamesobutler (2022-02-26 13:07 UTC)

<p>The OpenJPEG build error as part of the ITK project is because you’re likely using Visual Studio 2019 version 16.9 or later. This issue was fixed in <a href="https://github.com/InsightSoftwareConsortium/ITK/commit/7cebc26dfae73d90dbf4702aa1ccc544e9b70a37" rel="noopener nofollow ugc">this commit</a>, but that means you will need to build latest Slicer <code>master</code> branch to have that included. You were building Slicer 4.11.20200930 which is quite old and even older than the current Slicer 4.11.20210226.</p>
