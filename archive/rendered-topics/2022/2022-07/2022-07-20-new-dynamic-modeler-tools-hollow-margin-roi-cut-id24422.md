# New Dynamic Modeler tools: Hollow, Margin, ROI cut

**Topic ID**: 24422

## Post #1 by @lassoan (2022-07-20 21:39 UTC)

<p>Dynamic Modeler module allows parametric editing of models. In this post we present three new tools have been added to this module some time ago (by <a class="mention" href="/u/sunderlandkyl">@Sunderlandkyl</a> and <a class="mention" href="/u/lassoan">@lassoan</a>) but they have not been announced yet.</p>
<p><strong>Hollow tool:</strong> creates a hollow model (with a specified wall thickness) from a solid surface mesh.</p>
