## Post #2 by @Sunderlandkyl (2022-12-07 16:12 UTC)

<p>The NeuroSegmentation module is a module for manual segmentation using segment editor tools. It has some additional tools to make it easier to perform the segmentation, but no automated segmentation functionality.</p>
<p>There will be an associated guide published in the future for performing neuro segmentation using this module as part of the workflow, but it is not currently public.</p>
