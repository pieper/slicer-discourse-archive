## Post #1 by @xiaoqian (2025-05-23 11:10 UTC)

<p>PC(Mac OS ,3D slicer 5.8.1)<br>
My problem is that</p>
<p>I do not want to change the actual spatial information or rotate the image itself; instead, I simply want to redefine the default slice orientations so that the data is displayed in standard axial, coronal, and sagittal views—where the sagittal plane is perpendicular to the LR and PA axes, the coronal plane is perpendicular to the LR and IS axes, and the axial plane is parallel to the LR and PA axes.<br>
In 3D Slicer, I can use the “Reformat” module and select “axial”, “coronal”, or “sagittal” views to correctly display the image slices. However, these adjustments are not saved with the image, and when I load the image into other software, it still defaults to the original (incorrect) orientations.<br>
What I need is a way to <strong>redefine and save the default slice orientations</strong> so that the image is displayed with standard anatomical planes in other software as well.</p>
<p>The image1  is the original state of my data. I want to modify the slice angle of the image1 to the state shown in the second image and be able to save it in the file. Since I will be using other software later, I hope the second image becomes the default state.</p>
