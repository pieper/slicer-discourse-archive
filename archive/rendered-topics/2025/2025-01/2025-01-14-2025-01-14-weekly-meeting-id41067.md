<h4><a name="p-121554-refactor-corner-annotations-5" class="anchor" href="#p-121554-refactor-corner-annotations-5" aria-label="Heading link"></a>Refactor Corner Annotations</h4>
<ul>
<li>
<p>See <a href="https://github.com/Slicer/Slicer/issues/7184">Slicer Issue #7184</a>.</p>
</li>
<li>
<p><strong>Custom User Annotations</strong>:</p>
<ul>
<li>
<p>Users can implement custom annotations by <strong>deriving</strong> from <code>vtkMRMLAbstractAnnotationPropertyValueProvider</code>.</p>
<ul>
<li>This acts as a <strong>custom property provider</strong> for the XML parser.</li>
</ul>
<pre data-code-wrap="C++"><code class="lang-C++">virtual bool CanProvideValueForPropertyName(const std::string &amp;propertyName) = 0;
virtual std::string GetValueForPropertyName(const std::string &amp;propertyName,
                                        const XMLTagAttributes &amp;attributes,
                                        vtkMRMLSliceNode *) = 0;
virtual std::unordered_set&lt;std::string&gt; GetSupportedProperties() = 0;
</code></pre>
</li>
<li>
<p><strong>XML Annotation Structure</strong>:</p>
<ul>
<li>Custom attributes can be rendered in slice views using the following structure:</li>
</ul>
<pre data-code-wrap="XML"><code class="lang-XML">&lt;annotations&gt;
  &lt;corner position="bottom-left"&gt;
    &lt;property name="Label" prefix="L: " display-level="always"/&gt;
    &lt;property name="Foreground" prefix="F: " display-level="always"/&gt;
    &lt;property name="Background" prefix="B: " display-level="always"/&gt;
  &lt;/corner&gt;
  &lt;edge position="bottom"&gt;
    &lt;property name="Label" prefix="L: " display-level="always"/&gt;
  &lt;/edge&gt;
&lt;/annotations&gt;
</code></pre>
</li>
</ul>
</li>
<li>
