<li>Begin the release process on <strong>January 22nd</strong>.</li>
</ul>
</li>
</ul>

<ul>
<li><strong>Overview</strong>: <a class="mention" href="/u/arhowe00">@arhowe00</a> presented an overview of <a href="https://github.com/Slicer/Slicer/pull/8143">PR #8143</a>.</li>
<li><strong>Generalizing MRML Node Attributes</strong>:

</ul>
</li>
</ul>

<ul>
<li><strong>Simplifying Scene Creation</strong>:
<ul>
<li>Discussed facilitating the creation of SOFA scenes in Slicer-SOFA.   <a href="https://github.com/bakpaul">Paul Baksic</a> is developing a “Prefabs” system (<a href="https://github.com/Slicer/SlicerSOFA/issues/40">Issue Slicer/SlicerSOFA#40</a>).</li>

</ul>
</li>

<ul>

</ul>
