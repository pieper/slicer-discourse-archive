<p>The Slicer trame extension is now available in Slicer 5.10 and allows to create modern web applications while leveraging 3D Slicer’s core functionalities.</p>
<p>The extension is built on top of 3D Slicer python’s wrapping mechanism and the <a href="https://kitware.github.io/trame/" rel="noopener nofollow ugc">trame framework</a> and provides building blocks to create medical web application with server side processing and rendering.</p>
<p>For more information, don’t hesitate to visit Kitware’s blog post: <a href="https://www.kitware.com/trame-slicer-announcement/" rel="noopener nofollow ugc">https://www.kitware.com/trame-slicer-announcement/</a></p>
<p>Many thanks to the 3D Slicer community for making this extension possible!</p>
