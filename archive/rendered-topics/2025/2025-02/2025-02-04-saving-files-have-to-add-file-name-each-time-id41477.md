## Post #1 by @CTuser (2025-02-04 00:54 UTC)

<p>When performing multiple measurements on multiple files, I find myself saving data frequently. It would be very helpful if the same file is open, if the save name did not revert to the default after each save. In addition, if the scene contains multiple measurements each of these need to be individually labelled so they do not overwrite previous measurements. It would be great if we can specify a name that gets populated as a prefix across all measurements and as new measurements are added.</p>
