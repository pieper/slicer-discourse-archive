## Post #1 by @jcfr (2025-11-07 19:45 UTC)

<p>We’ve temporarily disabled regular Preview and Stable builds of Slicer (and associated extensions) while we prepare the Slicer 5.10 Stable release.</p>
<p>The Slicer repository will be tagged tomorrow, and the release process will start shortly afterward.</p>
<p>In parallel, we are:</p>
<ul>
<li>Finalizing the upgraded macOS build system (details <a href="https://discourse.slicer.org/t/maintenance-of-macos-dashboard-in-progress-upgrading-from-ventura-to-sequoia/44950/">here</a>)</li>
<li>Integrating infrastructure changes needed for future <strong>Qt6</strong> support</li>
</ul>
<p>Note that <strong>Slicer 5.10 will still ship with Qt 5.15</strong>. These infrastructure changes are forward-looking steps toward bundling Qt6 by default in a later release.</p>
<p>You can track progress here: <a href="https://github.com/Slicer/Slicer/issues/8828">https://github.com/Slicer/Slicer/issues/8828</a></p>
<p>Thanks for your patience <img src="https://emoji.discourse-cdn.com/twitter/pray.png?v=15" title=":pray:" class="emoji" alt=":pray:" loading="lazy" width="20" height="20"></p>
