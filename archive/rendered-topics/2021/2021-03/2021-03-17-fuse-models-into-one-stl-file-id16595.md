## Post #3 by @lassoan (2021-03-17 14:36 UTC)

<p>If models do not overlap then you can use “Merge models” module. If the models overlap then you can compute Boolean union operation using Combine models module - provided by Sandbox extension (you can install it using Extensions manager; the extension is in Examples category).</p>
