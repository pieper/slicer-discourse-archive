## Post #2 by @lassoan (2021-05-28 21:38 UTC)

<p>You cannot modify Annotation ROI control point sizes. However, in recent Slicer Preview Releases you can create markups ROIs and use that for cropping volumes. Markups ROIs have adjustable control points size and other advantages (e.g., they can be rotated).</p>
