## Post #1 by @muratmaga (2021-07-23 22:29 UTC)

<p>Please follow the instructions at <a href="https://github.com/SlicerMorph/SlicerMorph#installation" rel="noopener nofollow ugc">the official repository</a>.</p>
<p>We advise using the latest stable version of 3D Slicer with SlicerMorph, as updating SlicerMorph is much simpler that way. If you opt to use a preview release, remember that you will need to download and install a newer release to get the updates.</p>
