## Post #2 by @lassoan (2021-07-26 03:22 UTC)

<p>Note for interpreting the message above: SE = Segment Editor</p>
<p>There are many ways to create “smoother” cuts:</p>
<ul>
<li>You can create straight cut edges if you select circle or rectangle mode in Scissors effect.</li>
<li>You can apply <a href="https://discourse.slicer.org/t/new-segment-editing-feature-smoothing-brush/14577">local smoothing</a> (e.g., opening with kernel size of 3mm).</li>
<li>You can create a cutout shape using Baffle Planner module (in SlicerHeart extension) and then use that shape to cut out from the model using Combine models module (in Sandbox extension).</li>
<li>You can use Dynamic Modeler module’s “ROI Cut” tool to cut straight edges with a resizable, rotatable, rectangular ROI</li>
</ul>
