# SlicerCervicalSpine extension is now online

## Post #1 by @brhoom (2019-03-06 15:56 UTC)

<p>SlicerCervicalSpine extension is now online and can be installed via Extension Manager. This extension helps to provide an automatic approximated segmentation of the cervical spine from multi-modal images e.g. CT and MRI. It also detects the origin and insertion point of cervical spine automatically as well in addition to provide other information.</p>
<p>A video demo can be found <a href="https://www.youtube.com/channel/UC22K0PzsrWoThJwvJM56Okw" rel="noopener nofollow ugc">here</a>.</p>
