## Post #2 by @jcfr (2019-11-26 22:23 UTC)

<blockquote>
<p>Slicer 5.0 release schedule</p>
</blockquote>
<p>The plan is to release Slicer 5.0 shortly after the <a href="https://projectweek.na-mic.org/PW33_2020_GranCanaria/">33rd Project Week</a></p>
<p>We updated the <a href="https://www.slicer.org/wiki/Documentation/Labs/Slicer5-roadmap">Slicer roadmap</a> adding a new section called <code>Slicer 5.2: Backward incompatible changes</code> and retargeted some of the low priority tasks.</p>
<p>During the project week, we plan on:</p>
<ul>
<li>discussing how to improve the <a>MRML Copy API</a> and hopefully make good progress</li>
<li>implementing the following changes:
<ul>
<li>Update default <a href="https://www.slicer.org/wiki/Documentation/Labs/Slicer5-roadmap#Coordinate_system_in_files">Coordinate system in STL files</a></li>
<li>Update default for <a href="https://www.slicer.org/wiki/Documentation/Labs/Slicer5-roadmap#Acquisition_transform">Acquisition transform</a></li>
<li>Updating the <a href="https://www.slicer.org/wiki/Documentation/Labs/Slicer5-roadmap#Extension_description_file_format">Extension description file format</a></li>
</ul>
</li>
</ul>
