## Post #2 by @lassoan (2019-07-23 17:41 UTC)

<p>Yes, you can easily move all non-connected segment components to separate segments using Islands effect / Split islands to segments. To move only a selected island to another segment: create a new segment, activate Islands effect / Add selected island, and click on the island in a slice view (to find a point that you see in a 3D view, hold down Shift key while you move the mouse to the selected point).</p>
