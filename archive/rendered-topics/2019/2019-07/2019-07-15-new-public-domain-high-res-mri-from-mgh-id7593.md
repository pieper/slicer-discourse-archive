## Post #1 by @pieper (2019-07-15 15:17 UTC)

<p>There’s an amazing high res ex-vivo scan available for anyone interested in looking at the glories of the human brain.    Be prepared for a long download and some slow operations.  But once it’s loaded it’s really interesting to look at.  If your machine is reasonable in terms of ram most operations like slice pan/zoom/scroll are very interactive in all planes (I tested on a recent mac book air for example).</p>
<p>The <a href="https://datadryad.org/handle/10255/dryad.217211?show=full" rel="nofollow noopener">data is available here</a>, and a <a href="https://www.biorxiv.org/content/early/2019/05/31/649822.full.pdf" rel="nofollow noopener">paper preprint describing the acquisition</a> is also available.</p>
<p>The example videos below were made with the SceenCapture module (actual time for volume rendering is a few seconds per frame).</p>

## Post #2 by @pieper (2019-07-15 19:09 UTC)

<p>Here’s another video showing interactive volume rendering of the 100um isotopic MRI using 3D Slicer accessed through a web browser remote session to a cloud GPU using <a href="https://github.com/QIICR/SlicerGCPSetup" rel="nofollow noopener">the instructions here</a></p>
<p>Data size is 1760x1760x1280 floating point (just under 16g)</p>
<p>Hardware is a 8 core 30 gig of ram google cloud compute engine with attached v100 GPU with 16G of ram.</p>
