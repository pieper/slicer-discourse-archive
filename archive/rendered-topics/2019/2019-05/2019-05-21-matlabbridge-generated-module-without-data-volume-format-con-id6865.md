<p>Hi Guys,</p>
<p>We used matlabbridge to develop our own module. The method we have developed using matlab and only support .nii file. However, with the module generated by matlabbridge according to the tutorial, it seems that the module convert the data volume to “.nrrd” format first and then set the nrrd volume to the function. If so, is there any way to skip the conversion.<br>
Thanks!</p>

## Post #2 by @lassoan (2019-05-21 17:09 UTC)

<p>Slicer can write the image in .nii format as well, just change the file extension in the module .xml file from nrrd to nii.</p>
