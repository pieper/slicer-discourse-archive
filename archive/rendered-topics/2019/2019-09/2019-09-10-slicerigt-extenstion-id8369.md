## Post #2 by @Sam_Horvath (2019-09-10 18:51 UTC)

<p>We are currently having some instability with the factory build system, so the latest releases may not have all of the extensions.   We will have this fixed hopefully in the next few days. I recommend a slightly older nightly:</p>
<p><a href="http://slicer.cdash.org/index.php?project=SlicerPreview&amp;date=2019-09-05" class="onebox" target="_blank" rel="nofollow noopener">http://slicer.cdash.org/index.php?project=SlicerPreview&amp;date=2019-09-05</a></p>
<p>Under Nightly-packages you will see entries for the platforms, and a yellow package icon that will take you to a download link for the installer.</p>
<p>On the CDash instance, you can filter for the extension name for a particular data to confirm that it is available: <a href="http://slicer.cdash.org/index.php?project=SlicerPreview&amp;date=2019-09-05&amp;filtercount=1&amp;showfilters=1&amp;field1=buildname&amp;compare1=63&amp;value1=SlicerIGT" rel="nofollow noopener">http://slicer.cdash.org/index.php?project=SlicerPreview&amp;date=2019-09-05&amp;filtercount=1&amp;showfilters=1&amp;field1=buildname&amp;compare1=63&amp;value1=SlicerIGT</a></p>
<p>For details on the factory status, see: <a href="https://discourse.slicer.org/t/update-on-build-factory-status/8146" class="inline-onebox">Update on Build factory status</a></p>
