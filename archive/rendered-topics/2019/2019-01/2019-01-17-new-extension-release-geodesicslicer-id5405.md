## Post #1 by @Frederic (2019-01-17 18:19 UTC)

<p>Dear all,<br>
I am very proud to announce the release of new extension in 3D Slicer: GeodesicSlicer</p>
<p>This extension calculates geodesic path in 3D structure. Moreover, thanks to this geodesic path, this module can draw an EEG 10-20 system, determine the projected scalp stimulation site (MRI guided brain stimulation without the use of a neuronavigation System) and correct the rTMS resting motor threshold by correction factor.</p>
